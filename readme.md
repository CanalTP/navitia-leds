#navitia-leds

Real-time visualization of navitia journey requests

![Demo](http://i.giphy.com/d2Z5KWO8O237asX6.gif)

##Installation

Install Python dependencies
```pip install -r requirements.txt```

Get submodules
```git submodule update --init```

Setup project
```setup.py build_pbf```

Start the program
```manage.py settings.json```
